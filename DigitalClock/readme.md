# Digital Clock

## Features

* Displays the current time in real-time
* Updates automatically every second
* Supports 12-hour format with AM/PM
* Displays current day, month, date, and year
* Uses leading zeros for consistent time formatting

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## How It Works

* JavaScript creates a new Date object to get the current date and time.
* The time is updated every second using `setInterval()`.
* Hours, minutes, and seconds are formatted and displayed dynamically.
* Day and month numbers are converted into readable names using arrays.
* The clock content is updated in real-time using DOM manipulation.
