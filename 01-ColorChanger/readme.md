# Color Changer

## Features

* Change the background color with a button click
* Multiple color options available
* Instant UI updates
* Simple and interactive design

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## How It Works

* The application displays buttons for different colors.
* When a color button is clicked, the selected color is passed to a JavaScript function.
* JavaScript updates the page background using DOM manipulation.
* The background color changes instantly based on the selected button.
